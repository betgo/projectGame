# Prompt Library

Prompt files in this folder are versioned role contracts.

Rules:

1. File names must include version suffix (example: `BUILDER_v2.md`).
2. Commits must cite prompt IDs in `Prompt-Refs`.
3. Prompt updates must include a rationale in commit `Why`.

Current roles:

- ARCHITECT_v1
- PLANNER_v1
- BUILDER_v2
- GUARDIAN_v1
- REVIEWER_v1
